
LingoBox - نسخة تجريبية للمستخدم Ahmed
محتوى الحزمة:
- index.html : صفحة البداية
- styles.css : تصميم بسيط متجاوب
- app.js : منطق الواجهة (محاكاة تسجيل، اشتراك، رسائل، محادثة ذكية)
- README.txt : هذا الملف

تشغيل محلي:
1) فك الضغط (أو فتح الملف) ثم افتح /mnt/data/language_platform/index.html في المتصفح أو شغّل Live Server على مجلد المشروع.
2) هذا مشروع فرونت-إند تجريبي. لا توجد تكاملات حقيقية مع Google/OAuth أو بوابات دفع — كل شيء محاكى بالـ localStorage.
3) لاستخدام ميزات المحادثة أو الرسائل، اضغط "تسجيل / دخول" وقم بتسجيل اسم (محاكاة).
4) لاختبارات أسبوعية: النظام يفتح حزمة أسئلة جديدة كل أسبوع (محسوبة حسب الأسابيع منذ Epoch) وتُسجل تلقائياً.
5) الاشتراك: زر اشترك يقوم بتخزين حالة الاشتراك في localStorage (محاكاة).

ملحوظات حول المتطلبات الحقيقية:
- لربط Google/Facebook/Telegram OAuth تحتاج إلى إعدادات على منصاتهم وبيانات خادم (Client ID/Secret) — هذه لم تُنفّذ هنا.
- للمكالمات الحقيقية تحتاج WebRTC + خادم إشارة (signaling server) — غير متضمن.
- للمحادثة مع AI حقيقية تحتاج واجهة إلى خدمة مثل OpenAI أو خدمة مشابهة (API key) — هنا محاكاة.
